🚀 一键安装

专为Windows用户打造，无需任何技术背景，轻松部署你的AI助手

1

下载安装包

点击本页面按钮下载 OpenClaw-Setup.exe 安装包（约5MB）

2

运行安装程序

右键选择"以管理员身份运行"，或双击直接运行

3

等待自动安装

程序会自动完成 WSL、Ubuntu、Node.js 等所有环境的安装（约10-30分钟）

4

开始使用

安装完成后，点击桌面快捷方式即可启动 OpenClaw

✓ Windows 10/11
✓ 管理员权限
✓ 至少 10GB 可用空间
✓ 稳定的网络连接

❓ 常见问题

Q: 安装过程需要多久？

取决于你的网络速度和网络环境，通常需要 10-30 分钟。首次安装 WSL 可能需要重启电脑。

Q: 安装失败怎么办？

可以尝试使用"强制重装.bat"或"清除代理并安装.bat"脚本。也可以联系客服获取远程安装支持。

Q: 如何配置AI模型？

安装完成后，运行配置向导，可以选择使用 Anthropic、OpenAI 或本地模型，并填写相应的 API Key。

Q: 需要卸载怎么办？

可以在开始菜单中找到卸载选项，或运行安装目录下创建的卸载程序。
